
-------------------------------------------------
Cheatsheet CLI shortcuts:

$ CHEATSB   # open all cheatsheets in sublime
$ CHEATLS   # ls cheatsheet dir
$ CHEATCD   # cd into cheatsheet dir

$ BASH      # bash cli cheatsheet
$ GIT       # man git
$ NPM       # echo global npm modules dir
$ ND        # node debug cheatsheet
